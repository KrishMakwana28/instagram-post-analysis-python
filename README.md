# 📸 Instagram Post Analysis using Python

A comprehensive Exploratory Data Analysis (EDA) project that examines Instagram post performance using Python. The project focuses on understanding engagement patterns, posting behavior, and the factors that contribute to successful social media content.

---

## 📌 Project Overview

This project analyzes Instagram post data to uncover valuable insights into user engagement, content performance, and posting trends. Through data cleaning, visualization, and statistical analysis, it helps identify the characteristics of high-performing Instagram posts.

---

## 📂 Dataset (Instagram Data.csv)

The dataset contains Instagram post information such as:

- Post ID
- Caption
- Date & Time
- Likes
- Comments
- Engagement Metrics
- Hashtags
- Media Type
- Reach / Impressions (if available)
- Other post-related attributes
